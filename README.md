It took me a hot minute to figure out how to resize those images in Github Flavoured Markdown, so to practice code blocks, here is the solution that worked best for me: -
- `![Photo of Blackcap](https://github.com/user-attachments/assets/a9dda244-cbf2-442d-b9a9-71e719f67922)` Doesn't allow for resizing
- Instead, copy `<img src= "url" />`, and replace `![](url)`, then add width and height attributes, `width="200" height="200"`, like so:
 - ```<img src="https://github.com/user-attachments/assets/a9dda244-cbf2-442d-b9a9-71e719f67922" width="200" height="400" />```
